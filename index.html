Sample DCF Calculation (Code)


def discounted_cash_flow(cash_flows, discount_rate):
    """
    cash_flows: list of future cash flows (year 1, 2, 3, ...)
    discount_rate: annual discount rate (e.g., 0.10 for 10%)
    """
    return sum(
        cf / (1 + discount_rate) ** t
        for t, cf in enumerate(cash_flows, start=1)
    )

cash_flows = [1200, 1300, 1400, 1500]  # in million INR
r = 0.10  # 10% cost of capital

npv = discounted_cash_flow(cash_flows, r)
print(f"NPV of project: ₹{npv:,.2f} million")

Financial Formulas (Math)

Net Present Value (NPV):

\[ \text{NPV} = \sum_{t=1}^{N} \frac{CF_t}{(1 + r)^t} - C_0 \]

Compound Annual Growth Rate (CAGR):

\[ \text{CAGR} = \left(\frac{V_{\text{final}}}{V_{\text{initial}}}\right)^{\frac{1}{n}} - 1 \]
